# PDF to tag-cloud generator

## Installation

Run the following commands in terminal:

```
pip install virtualenv --user
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
```

## Running the project
```
python manage.py migrate
python manage.py runserver
```
